# nbra-initial-operational-baseline

## Question

What baseline operating model emerges from the first NBRA source batch for building a functioning Global Health Security wiki corpus?

## Findings

The current corpus supports a baseline model with four pillars: legal-policy alignment, source-to-action tooling, workforce training pipelines, and continuous operational updates.

## Lessons

- Sustainable implementation requires institutional ownership before tooling expansion.
- Tool inventories create value only when connected to execution and monitoring loops.
- Training assets should be treated as core infrastructure, not optional support material.
